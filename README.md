\# POS System Management

A professional desktop application built with \*\*C#\*\* and \*\*SQL Server\*\* using \*\*N-Tier Architecture\*\*.



\## Key Features

\* \*\*N-Tier Architecture:\*\* Clean separation between Business Logic (BLL) and Data Access (DAL).

\* \*\*Inventory Management:\*\* Full CRUD operations for products and categories.

\* \*\*Invoicing System:\*\* Generate and manage digital invoices and sales records.

\* \*\*Validation \& Security:\*\* Integrated validation classes and security helpers for data integrity.



\## Tech Stack

\* \*\*Language:\*\* C# (.NET Windows Forms)

\* \*\*Database:\*\* Microsoft SQL Server

\* \*\*Data Access:\*\* ADO.NET
